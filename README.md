# MultiToolBot 🤖

Многофункциональный Telegram бот с набором полезных инструментов и ИИ генерацией изображений.

## 📋 Функции

- ⏱️ **Таймер** - установка таймера на нужное количество секунд
- 🌐 **Переводчик** - перевод текста на русский, английский, немецкий, французский
- 📊 **Подсчеты** - подсчет слов и символов в тексте
- 🔐 **Генератор паролей** - создание надежных паролей
- 🎨 **ИИ генератор изображений** - создание изображений по текстовому описанию
- 🎮 **Игры** - кубики (d4, d6, d8, d10, d12, d20, d100) и монетка

## 🚀 Установка

### Требования
- Python 3.8 или выше
- Telegram Bot Token (получить у [@BotFather](https://t.me/BotFather))
